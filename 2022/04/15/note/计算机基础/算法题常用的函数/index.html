# 二、 常用函数及方法

# 1. 数据转换

  • JavaScript：
    • 转字符串： String() 、 .toString()
      • 数组转字符串： list.join('')
    • 转数字： Number() 、 parseInt()
    • 转数组： Array.from()
      • 字符串转数组： s.split("")

# 2. swap 函数

# 3. 排序

  • JavaScript：仅有数组可以， list.sort(function)
    • 如 list.sort((a,b)=>a-b) ：升序

# 4. 遍历

  • JavaScript：

    • 遍历数组

      以上性能依次递减。

    • 遍历 map 、 set ：使用 Array.from 转换 key-value 就行了

    • 遍历 object ：使用 Object.keys 、 Object.values 、 Object.entries 就行了，也可以用 for-in
