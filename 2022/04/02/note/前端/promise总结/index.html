# Promise 是什么？

Promise（期约）实际上就是个对象，或者也可以说是个构造函数。构造函数内定义了 resolve 、 reject 、 all 等方法，在其原型 Promise.prototype 内定义了 then 、 catch 、 finally 等方法。

一般 Promise 的对象实例有三种状态：

  • 待定（pending）： promise 初始态，等待接受任务
  • 兑现（fulfilled，或者 resolve）：解决状态，调用 resolve() 方法进入
  • 拒绝（reject）：拒绝状态，调用 reject() 进入，一般是当拒绝时（比如出错了）调用

期约的状态改变不可逆，一般我们都希望能够正常解决任务，而不是出错拒绝（promise 的 reject 类似同步代码抛出错误，虽然不会中断进程，但也相当于请求失败）

# Promise.prototype.then()

then() 函数是为期约实例添加 resolve 处理程序的主要方法。

  • 参数：resolve 的实现函数，并将该函数推进队列。实现函数的参数为 resolve 的参数
  • 返回：新的 promise 实例，可以通过参数函数 return 指定，没有 return 则返回 Promise { undefined } ，没有参数则返回 p 的副本
