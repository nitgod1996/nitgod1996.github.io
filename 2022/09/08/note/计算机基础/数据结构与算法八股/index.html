# 十大排序

# 冒泡排序

  1. 两重循环
  2. 每次循环比较 i 和 i-1 ，把更大的换到后面，遍历结束后最大的在最后。
  3. 第二次循环长度 - 1，以此类推

复杂度： n^2 ，空间 1

# 选择排序

  1. 两重循环
  2. 每次循环找出当前最小值，遍历一轮结束，与 “最前” 元素互换
  3. 第二次循环长度 - 1（j=i），以此类推

复杂度： n^2 ，空间 1

# 插入排序

  1. 两重循环
  2. 每次循环将当前元素与前面元素比较，如果小于前者则前者后移，继续和新的前者比较。
  3. 直到到达合适位置插入（每轮结束前面 i 个元素都是有序的）
  4. 第二次循环长度 - 1（j=i），以此类推

复杂度： n^2 ，空间 1

# 快速排序

  1. 分治法
  2. 取最前面的元素，暂存为 temp，把数组中所有小于该元素的放在左边，大于该元素的放在右边，该元素插在中间
  3. 分别对左右两部分执行相同操作

# 归并排序

  1. 分治思想
  2. 自顶向下→对于每个数组，将其均分成两个小数组，直到不能分
  3. 自底向上→将小数组两两合并（合并 2 个有序数组）
  4. 最终合成一个大数组

复杂度 nlogn

# 堆排序

  1. 将数组构建成堆，以大顶堆举例
    1. 自底向上遍历，对于每个元素，进行 “下沉” 操作，
    2. 即和左右孩子节点比较，如果小于任何一个孩子节点，与更大的那个孩子节点互换
    3. 直到大于任意孩子节点
  2. 维护堆：每次取出堆顶，将堆底元素插入堆顶，进行 “下沉”
  3. 方法 1：建立一个新数组，依次取出堆顶元素放入新数组，实现排序
  4. 方法 2：依次将堆顶元素放到 “新的” 最后，实现原地排序（每次循环最后节点往前挪 1）

复杂度 nlogn

# 计数排序

  1. 是开辟一块连续格子组成的空间，给数据进行存储。
  2. 将数组中的数字，依次读取，存入其值对应的下标中。
  3. 储存完成后，再按照空间的顺序，依次读取每个格子的数据，输出即可。

所以，计数排序要求排序的数据，必须是有范围的整数。

时间复杂度： max(nums) ，空间复杂度： max(nums)

（其实就是，新建个数组，把元素当数组下标）

# 桶排序

优化的计数排序

  1. 创建 m 个桶（数组），每个桶定义该桶的数据范围
  2. 遍历数组，进入对应的桶中
  3. 每个桶中的数据单独排序
  4. 依次取出桶中的数据

时间复杂度： nlog(n/m) （前提是桶内排序用快排）

桶排序的应用场景十分严苛，首先，数据应该分布比较均匀。
