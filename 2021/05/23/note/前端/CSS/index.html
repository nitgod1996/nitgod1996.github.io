# 二、 选择器

选择器 示例 示例说明
.class .intro 选择所有 class="intro" 的元素
#id #firstname 选择所有 id="firstname" 的元素
* * 选择所有元素
element p 选择所有 <p> 元素
element,element div,p 选择所有 <div> 元素和 < p > 元素
element element div p 选择 <div> 元素内的所有 < p > 元素
element>element div>p 选择所有父级是 <div> 元素的 <p> 元素
element+element div+p 选择所有紧接着 <div> 元素之后的 < p > 元素
:before p:before 在每个 <p> 元素之前插入内容
:after p:after 在每个 <p> 元素之后插入内容
:first-child p:first-child 指定只有当 <p> 元素是其父级的第一个子级的样式。
:last-child p:last-child 选择每个 p 元素是其父级的最后一个子级。
:nth-child(n) p:nth-child(2) 选择每个 p 元素是其父级的第二个子元素
:nth-last-child(n) p:nth-last-child(2) 选择每个 p 元素的是其父级的第二个子元素，从最后一个子项计数
  • id 选择器 ：通过 id 属性定位容器。语法是 #idName
  • 类选择器：通过 class 属性定位容器。语法是.className

<font color=red> 注：类选择器以 </font><font size=20 color=red> 空格 </font><font color=red > 为结束符，即第一个单词匹配成功即可。</font>

如 class="intro demo" 亦可被 .intro 选择到。但 class="intro-demo" 就不行

  • element 选择器：通过标签定位容器。语法是直接 label {}

  • * 选择器：对所有容器进行 CSS 设置，通常用于初始化

  • :before/:after 选择器：在容器前 / 后插入内容，相当于在该位置创建一个新的容器并插入

  • 并列选择器：同时选中多个选择器，用 , 隔开

  • 子选择器：选择指定父标签的指定子标签，用 空格 隔开。

    • 父子选择器可以不同，如 div .center #id1 ，表示 div 子元素中 class=center 的子元素中 id=id1 的元素

  • 位数选择器：匹配所有的指定特征的第 n 个标签。选择器：位数选择器

    • :first-child：各组下的第 1 个标签。如 .box:first-child 表示每个 class='box' 组下的第一个标签
    • :last-child：最后一个标签
    • :nth-child(n)：各组第 n 个标签
    • :nth-last-child(n)：各组倒数第 n 个标签

# 三、 属性

在选择器内通过属性定义样式

# 1. 常见属性

# 1.1 background 背景：

设置容器背景（简写模式），直接在后面填写各类型参数。也可分别设置每个背景值。一般都是简写

background:bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;

  • background-color：背景颜色。
    • yellow
    • #00ff00
    • rgb(255,0,255)
  • background-position：背景图像的位置。以左上点定位，横轴 / 纵轴分别定位。
    • left top ：第一个是水平位置， left right center 。第二个是垂直位置 top bottom center
    • x% y% ：水平百分比，垂直百分比。相对于父容器的尺寸百分比
    • xpos ypos ： pos 是单位，可以是 px 像素，也可以是 rem 等其他单位
    • inherit ：继承自父元素
  • background-size：背景图片的大小。也是可以 像素值 或 百分比
  • background-repeat：当背景图较小时如何重复，一般设 no-repeat 不重复
  • background-image 指定要使用的一个或多个背景图像。 url(图片的路径)
  • background-origin：
  • background-clip
  • background-attachment 设置背景图像是否固定或者随着页面的其余部分滚动。

# 1.2 text 文本：

文本属性，主要用于设置段落信息。如对齐方式、行高等

  • direction：文本方向
    • ltr 默认。文本方向从左到右。
    • rtl 文本方向从右到左。
  • color：文本颜色
  • line-height：行高
    • number ：设置数字，此数字会与当前的字体尺寸相乘来设置行间距。
    • length ：设置固定的行间距。
    • % ：基于当前字体尺寸的百分比行间距。
  • text-align：对齐方式。 left 、 right 、 center 、 justify （两端对齐）
  • vertical-align：垂直对齐方式
  • word-spacing：字间距
  • 其他

# 1.3 font 字体：

设置字体。字体样式、大小、颜色等。简写如 font:15px arial,sans-serif; ，亦可单独设置

  • font-size：字体大小
  • font-style：字体样式。
    • italic 斜体，表示使用文字的斜体。
    • oblique 倾斜，表示让没有斜体属性的文字倾斜。
  • font-weight：字体粗细
    • bold 定义粗体字符。
    • bolder 定义更粗的字符。
    • lighter 定义更细的字符。
    • 直接数字定义
  • font-family：字体类型，如 Times New Roman ，也可导入外部字体，需要先用 @font-face 定义

# 1.4 列表 ul 、 lo 、 li 属性：

用于设置列表的属性，一般是列表项标志 ● 1. 这种

  • list-style：简写
  • list-style-type：列表项标志类型
    • none ：没有标志
    • disc 默认。标记是实心圆
    • circle 标记是空心圆。
    • 更多
  • list-style-image：用图像作列表项标志 url('sqpurple.gif')
  • list-style-position：列表项标志的位置
    • inside 列表项目标记放在文本内。
    • outside 默认值。位于文本外（左侧）。

# 1.5 border 边框：

用于设置容器的边框线

  • border：简写属性，用于把针对四个边的属性设置在一个声明。如 border:5px solid red;
  • border-style：设置线的样式，此属性可以有一到四个值。分别按↑、→、↓、←的顺序
    • dotted 定义点状
    • dashed 定义虚线
    • solid 定义实线。
    • double 定义双线
      • border-style:dotted solid double dashed;
        • 上边框是点状
        • 右边框是实线
        • 下边框是双线
        • 左边框是虚线
  • border-width：边框线宽
  • border-color：颜色
  • border-bottom：单独设置下边框的简写
  • border-bottom-color：下边框颜色...
  • 各边框都可各自定义

# 1.6 margin 外边距 /padding 内边距：

​ margin 也称页边距，与父元素的间距。padding 是与子元素的间距

  • margin：同时设置四个间距
    • auto 浏览器计算下外边距。
    • length 比如像素、厘米等。默认值是 0px。
    • % 基于父元素的宽度的百分比
    • inherit 继承父元素
  • margin-bottom 设置元素的下外边距。
  • margin-left 设置元素的左外边距。
  • margin-right 设置元素的右外边距。
  • margin-top 设置元素的上外边距。
  • padding：同上

# 1.7 display 显示：

  • display 通常用于隐藏 / 按行显示 / 按列显示
    • none ：隐藏所有组件。
    • inline ：显示为内联元素，元素前后没有换行符。不识别宽高设置
    • block ：显示为块级元素，此元素前后会带有换行符。
    • inline-block ：显示为行内块元素

注意：当把父元素设置为 inline ，不会对子元素产生影响。所以如果父元素是 inline 而子元素是 block ，那么仍会换行。

  • visibility 也有隐藏功能
    • hidden ：隐藏组件，但该隐藏仅仅隐藏显示，空间布局是不变的（即隐藏后仍有一块空白）

此处首先先明确一下内联元素（inline element）和块元素（block element）的概念。（一条标签如 <div> 即是一个元素）

# 1.7.1 内联元素

​ 内联元素又名行内元素，和其对应的是块元素 (block element)，都是 html 规范中的概念。内联元素的显示，为了帮助理解，可以形象的称为 “文本模式”，即一个挨着一个，都在同一行按从左至右的顺序显示，不单独占一行。

任何不是块级元素的可见元素都是内联元素。其表现的特性是 “行布局” 形式，这里的 “行布局” 的意思就是说其表现形式始终以行进行显示。

内联元素有以下特点：

  1. 和其他元素都在一行上，遇到父级元素边界会自动换行
  2. 高、行高以及内外边距都不可以改变
  3. 宽度与内容一样宽，且不可改变
  4. 行内元素只能容纳文本或者其他行内元素
  5. 对于行内元素，需要注意的是：设置宽度 width 无效，设置高度无效，可以通过设置 line-height 来设置，设置 margin 只有左右有效，上下无效，设置 padding 只有左右有效，上下无效

常见的内联元素有：

  • a – 锚点
  • abbr – 缩写
  • acronym – 首字
  • b – 粗体 (不推荐)
  • bdo – bidi override
  • big – 大字体
  • br – 换行
  • cite – 引用
  • code – 计算机代码 (在引用源码的时候需要)
  • dfn – 定义字段
  • em – 强调
  • font – 字体设定 (不推荐)
  • i – 斜体
  • img – 图片
  • input – 输入框
  • kbd – 定义键盘文本
  • label – 表格标签
  • q – 短引用
  • s – 中划线 (不推荐)
  • samp – 定义范例计算机代码
  • select – 项目选择
  • small – 小字体文本
  • span – 常用内联容器，定义文本内区块
  • strike – 中划线
  • strong – 粗体强调
  • sub – 下标
  • sup – 上标
  • textarea – 多行文本输入框
  • tt – 电传文本
  • u – 下划线

# 1.7.2 块元素

块元素和内联元素的基本差异是块元素一般都从新行开始，相邻的块级元素将会在不同行显示。

块级元素的特点：

  1. 独霸一行，总是在新行上开始
  2. 宽度缺省是它父级元素的 100%，除非设定一个宽度
  3. 高度、行高、外边距、内边距都可以设置
  4. 可以容纳其他内联元素或者其他块元素

以下是常见的块级元素：

  • address – 地址
  • blockquote – 块引用
  • center – 举中对齐块
  • dir – 目录列表
  • div – 常用块级容易，也是 CSS layout 的主要标签
  • dl – 定义列表
  • fieldset – form 控制组
  • form – 交互表单
  • h1~h6 – 标题
  • hr – 水平分隔线
  • isindex – input prompt
  • menu – 菜单列表
  • noframes – frames 可选内容，（对于不支持 frame 的浏览器显示此区块内容
  • noscript – 可选脚本内容（对于不支持 script 的浏览器显示此内容）
  • ol – 有序表单
  • p – 段落
  • pre – 格式化文本
  • table – 表格
  • ul – 无序列表

# 1.7.3 行内块元素

inline-block。整合了块元素和内联元素的优点，有以下特点：

  1. 元素排列在一行
  2. 宽度默认由内容决定
  3. 元素间默认有间距
  4. 支持宽高、外边距、内边距的所有样式的设置

# 1.8 position 定位：

​ 指定了元素的定位类型。通常与 left 等定位属性配合使用

  • position

    • static：无特殊定位，遵循正常文档流，忽略 left、top、right、bottom 和 z-index 属性

    • relative：相对于正常位置 (static) 移动， left 等属性设置的是移动值。

    • fixed：相对浏览器窗口固定，即使窗口滚动也不会动。适用于做浮窗

    • absolute：绝对定位，其实是相对于父元素的坐标， left 等属性设置的是坐标值。

      1. 设置了 absolute 的元素如果存在有祖先元素设置了 position 属性为 relative 或者 absolute，则这时元素的定位对象为此已设置 position 属性的祖先元素。

      2. 如果并没有设置了 position 属性的祖先元素，则此时相对于 body 进行定位。

    • sticky：粘性定位，设置一个阈值，相对于浏览器窗口不会超出此阈值。可见示例

  • left 、 right 、 bottom 、 top ：设置距左、右、下、上的距离
    • auto
    • length ： px 或 rem 或其他
    • %
    • inherit
  • z-index：堆叠顺序。最大的在最上面

# 1.9 overflow 溢出设置：

​ 定义了当元素溢出容器时如何显示，如

  • overflow
    • visible 默认值。内容不会被修剪，会呈现在元素框之外。
    • hidden 内容会被修剪，并且其余内容是不可见的。
    • scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
    • auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。
    • inherit 继承

# 2. 不常见属性

# 动画：

​ 用于设置元素动画。需要先定义动画规则 @keyframes （关键帧）

  • animation：简写，语法如下

    animation: name duration timing-function delay iteration-count direction fill-mode play-state;。

    示例： animation:mymove 5s infinite;

  • animation-name：指定要绑定到选择器的动画名称

  • animation-duration：动画指定需要多少秒或毫秒完成。即一轮动画多少秒，0 表示不执行动画

  • animation-timing-function：设置动画将如何完成一个周期

    • linear 动画从头到尾的速度是相同的。
    • ease 默认。动画以低速开始，然后加快，在结束前变慢。
    • ease-in 动画以低速开始。
    • ease-out 动画以低速结束。
    • ease-in-out 动画以低速开始和结束。
    • cubic-bezier(n,n,n,n) 自定义。在 cubic-bezier 函数中自己的值。可能的值是从 0 到 1 的数值。

  • animation-delay：设置动画在启动前的延迟间隔。

  • animation-iteration-count：播放次数

    • n 一个数字，定义应该播放多少次动画
    • infinite 无限循环

  • 更多
