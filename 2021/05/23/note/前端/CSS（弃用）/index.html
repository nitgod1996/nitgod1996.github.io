# 一、 层叠、优先级和继承

# 1. 层叠（冲突）与优先级

抛出问题：当多个不同来源的样式作用到同一个元素上时，哪个生效？

这就是层叠，层叠决定了如何解决样式冲突，一般按以下规则

  1. 样式表的来源：你写的样式（作者样式表）> 浏览器默认样式（用户代理样式表）

  2. 选择器优先级： !important > 行内样式（style 属性）>id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签（元素）选择器 = 伪元素选择器 > 通用选择器

    选择器 示例 示例说明 权重
    id 选择器 # #id 选择所有 id="id" 的元素 100
    类选择器 . .intro 选择所有 class="intro" 的元素 10
    属性选择器 [] [attribute] 选择所有有 attribute 属性的元素 10
    伪类选择器 : :link 作用于未访问的链接元素 10
    标签选择器 p 选择所有 <p> 元素 1
    伪元素选择器 :: ::first-line 为每个 <p> 元素的第一行文本添加样式 1
    组合选择器 , > , + , ~ 0（符号权重为 0，但实际上是两个被组合的权重相加）
    通用选择器 * 选择所有元素 0

    更详细的选择器介绍放在后面

  3. 源码顺序：就是写在后面的优先级更高

大致规则是：先看是否用户定义 -> 是否有！important-> 是否是行内样式 -> 计算优先级 -> 看源码顺序

一些注意点：

  • 尽量不用 id 选择器
  • 尽量不适用 !important ：上面 2 个会使样式很难被覆盖，造成恶性循环

# 2. 继承

如果某个元素没有定义样式，那么它会继承其祖先元素的样式。但只有以下属性可以被继承

  • 文本相关：
    • color：文本颜色
    • font：字体
      • font-family：字体样式（楷体、宋体...）
      • font-size：字体大小
      • font-weight：字体宽度
      • font-variant：字体粗细
      • font-style：加粗、倾斜、下划线...
    • line-height：行高（可用来垂直居中）
    • letter-spacing
    • text-align：文本居中
    • text-indent
    • text-transform、white-space 以及 word-spacing
