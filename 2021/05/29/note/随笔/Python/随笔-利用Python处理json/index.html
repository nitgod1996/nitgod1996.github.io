0. json类

​ 在Python 的 json库中，共有四个方法。分别是：

json.load() # json文件输入流转为dict
json.loads() # 将字符串类型的json转为dict
json.dump() # 将dict类型的json写到文件输出流
json.dumps() # 将dict类型的json转为

1. 通过pandas读写json

pandas是Python数据处理的库，一般把数据转成DATAFrame或Series进行处理，但这样可能会破坏原有的json格式

前提：

  1. 引入外部库
import json
import pandas as pd

  1. 读取数据，保存为Dataframe。以.csv文件为例

    df = pd.read_csv('DataA09/demo.csv')

    输出如下：第1行和第1列均为索引，不视作数据内容

  2. 利用DataFrame.to_json转成json字符串

    df.to_json('path',orient='index')#返回str类型
#或者
json.loads(df.to_json('path',orient='table'))["data"]#返回map
#就能转成比较常见的json格式文件

    orient = 'index'有时候中文乱码，可以和下方例子一样，用json.loads()包裹，再str()转回字符串，就可以显示中文了。但是仅仅是str是中文，json文件还是乱码的

2. 以dict格式读取json

字典（map表）格式是最适合保存json文件的

以Python自带的文件输入输出流读取：

# 读取数据
with open('../dataYe/od_map.json','r') as f:
    djson = json.load(f)
    
# 写入 JSON 数据
with open('data.json', 'w') as f:
    json.dump(data, f)
