# 一、 简介

# 1. pandas 作用

​ pandas 是 Python 用于数据分析的库。常见的用法：

  1. 以 DataFrame 读入、输出各种格式的数据文件。
  2. 处理二维数组，如删掉一行，类似 Excel 的功能

# 2. pandas 安装及引入

  1. 安装 pandas

    pip install Pandas

  2. 导入 pandas 库

    import pandas as pd

# 3. pandas 主要数据类型：DataFrame 和 Series

在 pandas 中，dataframe 和 series 的主要区别是：

dataframe 保存的是多维数组，而 series 保存的是一维数组

其他包括性质、用法等都基本差不多。

df=pd.DataFrame([[1,2],[2,3]])
print(type(df[0]))#

# 二、 DataFrame

pandas 读入数据后一般以 DataFrame 类型存储。 DataFrame 是类似二维数组的格式

# 1. 导出数据

以 DataFrame 导出到数据文件，设 df 是 DataFrame 类型，如下所示

第 1 行和第 1 列分别是行索引和列索引，不视作数据内容

  • 导出 json

    df.to_json('path',orient='index')
#`orient`：导出的json格式，有以下取值

    • 'index' ：最常用的就是这个，导出格式为

      {
  "0":{"name":"zhan","age":16,"height":120},
  "1":{"name":"li","age":17,"height":170},
  "2":{"name":"wang","age":20,"height":160}
}

    • 'split' ：

      {
  "columns":["name","age","height"],
  "index":[0,1,2],
  "data":[["zhan",16,120],["li",17,170],["wang",20,160]]
}

    • 'table' ： data 部分和 'index' 差不多

      {
  "schema":{
    "fields":[
      {"name":"index","type":"integer"},
      {"name":"name","type":"string"},
      {"name":"age","type":"integer"},
      {"name":"height","type":"integer"}
    ], 
    "primaryKey":["index"], 
    "pandas_version":"0.20.0"
  },
  "data":[
    {"index":0,"name":"zhan","age":16,"height":120},
    {"index":1,"name":"li","age":17,"height":170},
    {"index":2,"name":"wang","age":20,"height":160}
  ]
}

    • 不写的话是以下格式：和 'split' 略微有些相似

      {
  "name":{"0":"zhan","1":"li","2":"wang"},
  "age":{"0":16,"1":17,"2":20},
  "height":{"0":120,"1":170,"2":160}
}

  • 导出 csv：

    df.to_csv('path',index='False')
#index:是否把索引写进csv。默认是true

# 2. 导入数据

以 pandas.to_** 导入，导入后是 DataFrame 的类型存放

  • 导入 CSV， df 是 DataFrame 类型

    df=pd.to_csv('path',header=0,index_col=0)
#`header`:列名。header=0表示首行作为列名（默认）；header=[1,2]表示1,2行作为列名
#`index_col`:行索引。index_col=0表示首列作为行索引。

    索引是不能以 df.loc[0] 访问到的，索引被认为不是数组内容

  • 导入 json

    df = pd.read_json('path', orient='table')
#`orient`：json格式，如何导出就如何导入

# 3. 类型转换

  1. 其他类型转 DataFrame
df=pd.DataFrame(***)#***是被转换的对象

  1. DataFrame 转 numpy.ndarray （numpy 也是类似数组的数据类型）

    array=df.values

# 4. 查询检索

  1. 按索引号查询， .iloc[]

    data=df.iloc[]#里面填index号，从0开始
df.iloc[0] #返回0 1 2 3
df.iloc[:,0] #返回0 4 8 12
df.iloc[0,0] #返回0

  2. 按索引名查询， .loc[]

    data=df.loc[]#里面填index_name
df.loc['a'] #返回0 1 2 3
df.loc[:,'A'] #返回0 4 8 12

    注：索引名如果是 str ，则需要加上 '' 。如果是 integer ，则不能加 ''

  3. [] ，二维：按索引名返回一维序列。一维：按索引号或索引号返回对应元素

    df['A']#返回0 4 8 12
df['A'][0] #返回0
df['A']['a']#返回0

# 5. 计数

  1. .value_counts() ：归类计数。记录每个相同元素的数量，也可以得出队列中共有多少不同值 / 重复值

    pd.DataFrame([0,1,0,2]).value_counts()
#输出 series类型
0 2
2 1
1 1
