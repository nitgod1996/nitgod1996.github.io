一、 JavaScript简介

  1. HTML 定义了网页的内容
  2. CSS 描述了网页的布局
  3. JavaScript 控制了网页的行为，主要用于与用户的交互及提高用户体验。

1. JavaScript简单功能

  • 直接网页输出流

    //直接打印在网页上
document.write("<h1>这是一个标题</h1>");
document.write("<p>这是一个段落。</p>");
//以弹框弹出
window.alert("弹框");//或者直接alert也可

  • 对事件的响应

    <button type="button" onclick="alert('欢迎!')">点我!</button>
<!--在onclick中写js代码-->

  • 修改html内容/CSS样式

    x=document.getElementById("demo");  //通过ID查找元素（类似CSS选择器的作用，元素即是标签）
x.innerHTML="Hello JavaScript";    //改变内容（注入html）
x.style.color="#ff0000";           //改变样式

  • 输出到控制台（适用于用编辑器调试）

     console.log() //相当于java的system.out.println

2. 引入JavaScript

2.1 内嵌入html

直接在html中添加<script>标签，在<script>中写js代码。<script>引入位置会影响执行顺序，一般视情况放在head或body中

1.4 关键字let

let是ES6新引入的关键字。let 声明的变量只在 let 命令所在的代码块内有效。在 ES6 之前，JavaScript 只有两种作用域： 全局变量 与 函数内的局部变量。

var和let的区别是，**let是局部变量，而var则有点模糊（循环体内全局变量，方法体内局部变量）。所以在块内一般使用let创建对象**。

1.5 关键字const

const 声明一个只读的常量，一旦声明，常量的值就不能改变。

  • const 可用于声明一个或多个常量，声明时必须进行初始化，且初始化后值不可再修改：
const PI = 3.141592653589793;
PI = 3.14;      // 报错
PI = PI + 10;   // 报错
  • const对引用对象如数组等并不能锁死。可利用对应函数修改，但不能用=重新赋值
  • 不能使用 const 关键字来重置 var 和 let关键字声明的变量:
var x = 2;         // 合法
const x = 2;       // 不合法

4. RegExp正则表达式

JavaScript中正则表达式非常重要，许多函数需要使用正则表达式进行匹配。此外，JavaScript判断字符是数字还是字母也只能通过正则表达式。（JavaScript没有像Java一样的isAlpha等方法）

4.1 什么是正则表达式

正则表达式是由一个字符序列形成的搜索模式。

正则表达式可用于所有文本搜索和文本替换的操作。

4.2 语法

/正则表达式主体/修饰符(可选)

如var patt = /runoob/i

/runoob/i 是一个正则表达式。

runoob 是一个正则表达式主体 (用于检索)。

i 是一个修饰符 (搜索不区分大小写)。

4.3 匹配规则

4.3.1 字符区间匹配

表达式 描述
[abc] 查找方括号之间的任何字符。
[^abc] 查找任何不在方括号之间的字符。
[0-9] 查找任何从 0 至 9 的数字。
[a-z] 查找任何从小写 a 到小写 z 的字符。
[A-Z] 查找任何从大写 A 到大写 Z 的字符。
[A-z] 查找任何从大写 A 到小写 z 的字符。
[adgk] 查找给定集合内的任何字符。
[^adgk] 查找给定集合外的任何字符。
(red|blue|green) 查找任何指定的选项。

4.3.2 特殊字符匹配

元字符 描述
. 查找单个字符，除了换行和行结束符。
\w 查找数字、字母及下划线。
\W 查找非单词字符。
\d 查找数字。
\D 查找非数字字符。
\s 查找空白字符。
\S 查找非空白字符。
\b 匹配单词边界。
\B 匹配非单词边界。
\0 查找 NULL 字符。
\n 查找换行符。
\f 查找换页符。
\r 查找回车符。
\t 查找制表符。
\v 查找垂直制表符。

4.3.3 字符数量匹配

量词 描述
n+ 匹配任何包含至少一个 n 的字符串。 例如，/a+/ 匹配 “candy” 中的 “a”，”caaaaaaandy” 中所有的 “a”。
n* 匹配任何包含零个或多个 n 的字符串。 例如，/bo*/ 匹配 “A ghost booooed” 中的 “boooo”，”A bird warbled” 中的 “b”，但是不匹配 “A goat grunted”。
n? 匹配任何包含零个或一个 n 的字符串。 例如，/e?le?/ 匹配 “angel” 中的 “el”，”angle” 中的 “le”。
n{X} 匹配包含 X 个 n 的序列的字符串。 例如，/a{2}/ 不匹配 “candy,” 中的 “a”，但是匹配 “caandy,” 中的两个 “a”，且匹配 “caaandy.” 中的前两个 “a”。
n{X,} X 是一个正整数。前面的模式 n 连续出现至少 X 次时匹配。 例如，/a{2,}/ 不匹配 “candy” 中的 “a”，但是匹配 “caandy” 和 “caaaaaaandy.” 中所有的 “a”。
n{X,Y} X 和 Y 为正整数。前面的模式 n 连续出现至少 X 次，至多 Y 次时匹配。 例如，/a{1,3}/ 不匹配 “cndy”，匹配 “candy,” 中的 “a”，”caandy,” 中的两个 “a”，匹配 “caaaaaaandy” 中的前面三个 “a”。注意，当匹配 “caaaaaaandy” 时，即使原始字符串拥有更多的 “a”，匹配项也是 “aaa”。
n$ 匹配任何结尾为 n 的字符串。
^n 匹配任何开头为 n 的字符串。
?=n 匹配任何其后紧接指定字符串 n 的字符串。
?!n 匹配任何其后没有紧接指定字符串 n 的字符串。

4.4 修饰符

修饰符 描述
i 执行对大小写不敏感的匹配。
g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。
m 执行多行匹配。

    for (let i in nums)
    console.log(nums[i]);//i为索引

    nums.forEach(function (item)&#123;
    console.log(item);//item是nums的元素，而非索引
&#125;);

  4. for of(需要ES6支持，性能优于forin，劣于普通for)

    for (let value of nums)
    console.log(value);//value是元素，而非索引

  5. map遍历(性能不及forEach)

    nums.map(function (num)&#123;
    console.log(num);//num是元素，而非索引
&#125;)

四、 DOM

DOM (文档对象模型)。通过 HTML DOM，可访问 JavaScript HTML 文档的所有元素。最主要的就是2点：

  1. 设置html文档内容
  2. 设置事件

在 HTML DOM (Document Object Model) 中 , 每一个元素都是 节点，或者说是对象:

  • 文档是一个文档节点。即document
  • 所有的HTML元素都是元素节点。即element，通过document获取
  • 所有 HTML 属性都是属性节点。即attr，在element中设置
  • 文本插入到 HTML 元素是文本节点。are text nodes。
  • 注释是注释节点。

大部分用到的节点都是元素对象和属性对象

1. document对象（节点）

当浏览器载入 HTML 文档, 它就会成为 Document 对象。

Document 对象是 Window 对象的一部分，可通过 window.document 属性对其进行访问。

1.1 通过document获取element对象

element对象，即html中的元素，也就是标签。一条标签对应一个element对象。

可以像css一样通过class/id/标签/name等选择器获取对象。

方法 描述
document.getElementsByClassName() 根据定义的class返回。以数组返回
document.getElementsByName() 根据定义name返回。以数组返回
document.getElementsByTagName() 根据标签名返回。以数组返回
document.getElementById() 返回对拥有指定 id 的第一个对象的引用。

如：

<div id="1" class="cls">哈哈</div>
<div>aa</div>
<textarea name="name1">文本框</textarea>
<script>
    document.getElementById("1");
    document.getElementsByClassName("cls");
    document.getElementsByTagName("div");
    document.getElementsByName("name1");
</script>

1.2 创建DOM节点对象

方法 描述
document.createAttribute("attribute") 创建一个属性节点 可用于设置标签的属性
document.createComment("text") 创建注释节点。
document.createElement("nodename") 创建元素节点。 nodename是标签名，或者input的type
document.createTextNode() 创建文本节点。

例：

var btn=document.createElement("BUTTON");
var t=document.createTextNode("CLICK ME");
btn.appendChild(t);

1.3 直接获取/设置html文档信息

可直接通过下列方法进行修改文档信息，常用方法如下

方法 描述
document.body 返回文档的body元素
document.title 返回当前文档的标题。
document.URL 返回文档完整的URL
document.links 返回对文档中所有 Area 和 Link 对象引用。
document.images 返回对文档中所有 Image 对象引用。
document.forms 返回对文档中所有 Form 对象引用。
document.inputEncoding 返回用于文档的编码方式（在解析时）。
document.lastModified 返回文档被最后修改的日期和时间。
document.anchors 返回对文档中所有 Anchor 对象的引用。
document.cookie 设置或返回与当前文档有关的所有 cookie。
document.documentURI 设置或返回文档的位置
document.domain 返回当前文档的域名。
document.write() 向文档写 HTML 表达式 或 JavaScript 代码。
document.writeln() 等同于 write() 方法，不同的是在每个表达式之后写一个换行符。

2. element对象（节点）

对应html的一条标签。

element对象可通过document对象获取

element对象可以完成以下功能：

  1. 设置/添加/移除属性
  2. 添加/移除子元素（节点）
  3. 修改html内容（一般是对对应标签的修改，而非像document那么粗暴直接修改body）
  4. 修改css样式

2.1 获取/设置/删除属性attr

方法 描述
element.attributes 返回一个元素的属性数组。可按索引访问attr
element.getAttribute(“attributename“) 根据属性名返回元素的属性值
element.getAttributeNode(“attributename“) 根据属性名返回指定属性节点对象（其实就是属性对象attr）
element.hasAttribute(“attributename“) 如果元素中存在指定的属性返回 true，否则返回false。
element.hasAttributes() 如果元素有任何属性返回true，否则返回false。
element.removeAttribute() 从元素中删除指定的属性
element.removeAttributeNode() 删除指定属性节点并返回移除后的节点。
element.setAttribute() 设置或者改变指定属性并指定值。
element.setAttributeNode() 设置或者改变指定属性节点。

2.2 设置标签体内容

方法 描述
element.innerHTML=text 设置或者返回元素的文本内容。
element.id 设置或者返回元素的 id。
element.nodeName 返回元素的标记名（大写） 文本以#text返回
element.nodeType 返回元素的节点类型
element.nodeValue 返回元素的节点值
element.href 返回元素的链接
element.style 设置或返回元素的样式属性
element.tagName 作为一个字符串返回某个元素的标记名（大写）文本以undefined返回
element.textContent 设置或返回一个节点和它的文本内容
element.title 设置或返回元素的title属性
element.toString() 一个元素转换成字符串
element.className 设置或返回元素的class属性
