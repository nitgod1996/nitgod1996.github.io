建议 jquery 的代码写在 $(document).ready 中，这是一个 jquery 的环境，也就是说在他中的代码可以被 jquery 识别，为了防止 jquery 库出现错误

# 一、 基本用法

# 1. 简介

jQuery 是一个轻量级的 "写的少，做的多" 的 JavaScript 库。极大地简化了 JavaScript 编程。

此外，jQuery 所有的功能函数均是针对元素的，也就是标签

jQuery 库包含以下功能：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript 特效和动画
  • HTML DOM 遍历和修改
  • AJAX
  • Utilities

提示： 除此之外，jQuery 还提供了大量的插件。

# 2. 安装 / 导入

  • Production version - 用于实际的网站中，已被精简和压缩。
  • Development version - 用于测试和开发（未压缩，是可读的代码）

以上两个版本都可以从 jquery.com 中下载。

# 3. 语法

通过 jQuery，您可以选取（查询，query） HTML 元素，并对它们执行 "操作"（actions）。

基础语法： $(selector).action()

  • 美元符号定义 jQuery
  • 选择符（selector）定位 HTML 元素，规则和 CSS 选择器相同
  • jQuery 的 action () 执行对元素的操作

实例:

  • $(this).hide () - 隐藏当前元素
  • $("p").hide () - 隐藏所有 <p> 元素
  • $("p.test").hide () - 隐藏所有 class="test" 的 <p> 元素
  • $("#test").hide () - 隐藏 id="test" 的元素

# 3.1 文档就绪 document ready

为了防止文档在完全加载（就绪）之前运行 jQuery 代码，即在 DOM 加载完成后才可以对 DOM 进行操作。如果在文档没有完全加载之前就运行函数，操作可能失败。

所以所有 jQuery 函数需要用 document ready 函数包裹

# 3.2 $() 返回类型

  • jQuery 获取的也是元素集合，集合中的元素可能是一个，也可能是多个。

  • jQuery 的函数是针对元素集合的。（类似字符串，"s" 也算字符串）

  • jQuery 获取的元素不能调用 JavaScript 的原生函数

  • 可以通过 $()[i] 获取 element 对象。可以调用 JavaScript 的 element 的函数，但可能就无法使用 jQuery 的函数了

# 6. jQuery 名称冲突

jQuery 使用 $ 符号作为 jQuery 的简介方式。

某些其他 JavaScript 库中的函数（比如 Prototype）同样使用 $ 符号。

jQuery 使用名为 noConflict() 的方法来解决该问题。

# 二、 jQuery 函数

jQuery 函数大多有以下参数，且含义接近：

  • speed：一般为函数执行速度。可选项为
    • 毫秒值
    • "slow"
    • "fast"
  • easing：函数执行效果。
    • "swing" - 在开头 / 结尾移动慢，在中间移动快
    • "linear" - 匀速移动
    • 扩展插件中提供更多可用的 easing 函数。
  • callback：回调函数。函数执行完之后，要执行的函数。

# 三、 AJAX

https://www.w3school.com.cn/jquery/jquery_ref_ajax.asp

AJAX 是与服务器交换数据的技术，它在不重载全部页面的情况下，实现了对部分网页的更新。

通过 ajax 可以实现前端页面调用后端数据

  1. 网页中发生一个事件（页面加载、按钮点击）
  2. 由 JavaScript 创建 XMLHttpRequest 对象
  3. XMLHttpRequest 对象向 web 服务器发送请求
  4. 服务器处理该请求
  5. 服务器将响应发送回网页
  6. 由 JavaScript 读取响应
  7. 由 JavaScript 执行正确的动作（比如更新页面）

# 1. 什么是 AJAX

AJAX = 异步 JavaScript 和 XML（Asynchronous JavaScript and XML）。

简短地说，在不重载整个网页的情况下，AJAX 通过后台加载数据，并在网页上进行显示。

同步：一定要等任务执行完了，得到结果，才执行下一个任务。

异步：不等任务执行完，直接执行下一个任务。（即两个不互相影响的任务并行）

​ 通过 jQuery AJAX 方法，您能够使用 HTTP Get 和 HTTP Post 从远程服务器上请求文本、HTML、XML 或 JSON - 同时您能够把这些外部数据直接载入网页的被选元素中。

# 3. jQuery、JavaScript 与 ajax 的关系

简单总结：

1、JS 是一门前端语言。

2、Ajax 是一门技术，它提供了异步更新的机制，使用客户端与服务器间交换数据而非整个页面文档，实现页面的局部更新。

3、jQuery 是一个框架，它对 JS 进行了封装，使其更方便使用。jQuery 使得 JS 与 Ajax 的使用更方便

关系比喻：

若把 js 比作木头，那么 jquery 就是木板（半成品）

jQuery 与 ajax 都是 js 的一个框架，各有各的功能，若 js 是父亲的话，jquery 与 ajax 就是两个儿子
