一、 基本用法

1. 简介

jQuery 是一个轻量级的”写的少，做的多”的 JavaScript 库。极大地简化了JavaScript编程。

此外，jQuery所有的功能函数均是针对元素的，也就是标签

jQuery 库包含以下功能：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript 特效和动画
  • HTML DOM 遍历和修改
  • AJAX
  • Utilities

提示： 除此之外，jQuery 还提供了大量的插件。

2. 安装/导入

有两个版本的 jQuery 可供下载：

  • Production version - 用于实际的网站中，已被精简和压缩。
  • Development version - 用于测试和开发（未压缩，是可读的代码）

以上两个版本都可以从 jquery.com 中下载。

2.1 本地导入

jquery-1.10.2.min.js文件下载好后，放到指定目录

<script src="jquery-1.10.2.min.js"></script>

2.2 在线导入

直接通过CDN(内容分发网络)引用。需要连网

<script src="https://cdn.staticfile.org/jquery/1.10.2/jquery.min.js">

3. 语法

通过 jQuery，您可以选取（查询，query） HTML 元素，并对它们执行”操作”（actions）。

基础语法： $(selector).action()

  • 美元符号定义 jQuery
  • 选择符（selector）定位 HTML 元素，规则和CSS选择器相同
  • jQuery 的 action() 执行对元素的操作

实例:

  • $(this).hide() - 隐藏当前元素
  • $(“p”).hide() - 隐藏所有

    元素

  • $(“p.test”).hide() - 隐藏所有 class=”test” 的

    元素

  • $(“#test”).hide() - 隐藏 id=”test” 的元素

3.1 文档就绪document ready

为了防止文档在完全加载（就绪）之前运行 jQuery 代码，即在 DOM 加载完成后才可以对 DOM 进行操作。如果在文档没有完全加载之前就运行函数，操作可能失败。

所以所有 jQuery 函数需要用document ready 函数包裹

$(document).ready(function()&#123;
 
   // 开始写 jQuery 代码...
 
&#125;);

或者使用简介点的写法，如下

$(function()&#123;
 
   // 开始写 jQuery 代码...
 
&#125;);

二、 jQuery函数

jQuery函数大多有以下参数，且含义接近：

  • speed：一般为函数执行速度。可选项为
    • 毫秒值
    • “slow”
    • “fast”
  • easing：函数执行效果。
    • “swing” - 在开头/结尾移动慢，在中间移动快
    • “linear” - 匀速移动
    • 扩展插件中提供更多可用的 easing 函数。
  • callback：回调函数。函数执行完之后，要执行的函数。

三、 AJAX

AJAX 是与服务器交换数据的技术，它在不重载全部页面的情况下，实现了对部分网页的更新。

通过ajax可以实现前端页面调用后端数据

1. 什么是AJAX

AJAX = 异步 JavaScript 和 XML（Asynchronous JavaScript and XML）。

简短地说，在不重载整个网页的情况下，AJAX 通过后台加载数据，并在网页上进行显示。

同步：一定要等任务执行完了，得到结果，才执行下一个任务。

异步：不等任务执行完，直接执行下一个任务。（即两个不互相影响的任务并行）

​ 通过 jQuery AJAX 方法，您能够使用 HTTP Get 和 HTTP Post 从远程服务器上请求文本、HTML、XML 或 JSON - 同时您能够把这些外部数据直接载入网页的被选元素中。

2. 语法

$.ajax({name:value, name:value, … })

ajax() 方法用于执行 AJAX（异步 HTTP）请求。

所有的 jQuery AJAX 方法都使用 ajax() 方法。该方法通常用于其他方法不能完成的请求。

常用参数如下：

名称 值/描述
async 布尔值，表示请求是否异步处理。默认是 true。
（同步请求将会锁住浏览器，用户其他操作必须等待请求完成才可以执行）
data 规定要发送到服务器的数据。
dataType 预期的服务器响应的数据类型。
error(xhr,status,error) 如果请求失败要运行的函数。
success(result,status,xhr) 当请求成功时运行的函数。拿到后端数据后主要操作就写在这里>
timeout 设置本地的请求超时时间（以毫秒计）。
type 规定请求的类型（**GET **或 POST）。
url 规定发送请求的 URL。默认是当前页面。

其中：

  • success：
    • result：请求到的数据
    • xhr： 包含 XMLHttpRequest 对象
    • status：响应状态码
  • error：
    • error：错误信息
    • xhr： 包含 XMLHttpRequest 对象
    • status：响应状态码

例：

$.ajax(&#123;
        type : "get",
        async : false,     //异步请求（同步请求将会锁住浏览器，用户其他操作必须等待请求完成才可以执行）
        url : "data/data1",    //服务器端放出api的地址，一般是把数据以json格式放出
        data : &#123;&#125;,
        dataType : "json",        //返回数据形式为json
        success : function(result) &#123;
            //请求成功时执行该函数内容，result即为服务器返回的json对象
            &#125;
        &#125;,
        error : function() &#123;//请求失败时执行该函数
            alert("图表请求数据失败!status"+status);
            myChart.hideLoading();
        &#125;
    &#125;)

3. 简介方法

ajax提供了一些简单的方法，直接获取指定类型的数据，不需要写参数居多的$.ajax

如$.getJSON()

$.getJSON("http://localhost:5000/data/data1",function (result)&#123;
        document.write(result) //result是获取到的json数据
    &#125;)
方法 描述
$.get() 使用 AJAX 的 HTTP GET 请求从服务器加载数据
$.getJSON() 使用 HTTP GET 请求从服务器加载 JSON 编码的数据
$.getScript() 使用 AJAX 的 HTTP GET 请求从服务器加载并执行 JavaScript

以$.getJSON()为例，

  • $.getJSON()：

    语法：$(selector).getJSON(url,data,success(result,status,xhr))

    • url：必需。规定将请求发送到哪个 URL。
    • data： 可选。规定发送到服务器的数据。
    • *success(result,status,xhr)*：success函数
