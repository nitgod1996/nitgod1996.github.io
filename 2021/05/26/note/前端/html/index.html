一、 简介

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>菜鸟教程(runoob.com)</title>
</head>
<body>
 
<h1>我的第一个标题</h1>
 
<p>我的第一个段落。</p>
 
</body>
</html>
  • 声明为 HTML5 文档
  • 元素是 HTML 页面的根元素
  • 元素包含了文档的元（meta）数据，如 定义网页编码格式为 utf-8。
  • 元素描述了文档的标题
  • 元素包含了可见的页面内容

  •  元素定义一个大标题

  •  元素定义一个段落

二、 标签

1. 标签类型

标签，也称元素，分为内联元素和块级元素。

块级元素会换行，而内联元素不会，可以通过CSS的display属性修改设置。

1.1 内联元素

​ 内联元素又名行内元素，和其对应的是块元素(block element)，都是html规范中的概念。内联元素的显示，为了帮助理解，可以形象的称为“文本模式”，即一个挨着一个，都在同一行按从左至右的顺序显示，不单独占一行。

任何不是块级元素的可见元素都是内联元素。其表现的特性是“行布局”形式，这里的“行布局”的意思就是说其表现形式始终以行进行显示。

内联元素有以下特点：

  1. 和其他元素都在一行上，遇到父级元素边界会自动换行
  2. 高、行高以及内外边距都不可以改变
  3. 宽度与内容一样宽，且不可改变
  4. 行内元素只能容纳文本或者其他行内元素
  5. 对于行内元素，需要注意的是：设置宽度width无效，设置高度无效，可以通过设置line-height来设置，设置margin只有左右有效，上下无效，设置padding只有左右有效，上下无效

常见的内联元素有：

  • a – 锚点
  • abbr – 缩写
  • b – 粗体(不推荐)
  • big – 大字体
  • br – 换行
  • cite – 引用
  • font – 字体设定(不推荐)
  • i – 斜体
  • img – 图片
  • input – 输入框
  • label – 表格标签
  • select – 项目选择
  • small – 小字体文本
  • sub – 下标
  • sup – 上标
  • textarea – 多行文本输入框
  • u – 下划线

1.2 块元素

块元素和内联元素的基本差异是块元素一般都从新行开始，相邻的块级元素将会在不同行显示。

块级元素的特点：

  1. 独霸一行，总是在新行上开始
  2. 宽度缺省是它父级元素的100%，除非设定一个宽度
  3. 高度、行高、外边距、内边距都可以设置
  4. 可以容纳其他内联元素或者其他块元素

以下是常见的块级元素：

  • address – 地址
  • center – 举中对齐块
  • dir – 目录列表
  • div – 常用块级容易，也是CSS layout的主要标签
  • dl – 定义列表
  • form – 交互表单
  • h1~h6 – 标题
  • hr – 水平分隔线
  • menu – 菜单列表
  • ol – 有序表单
  • p – 段落
  • table – 表格
  • ul – 无序列表

2. 基本标签

2.1 头部

<head> 元素包含了所有的头部标签元素。在 <head>元素中你可以插入脚本（scripts）, 样式文件（CSS），及各种meta信息。

可以添加在头部区域的元素标签为: <title>, <style>, <meta>, <link>, <script>, <noscript> 和 <base>。

标签 描述
<head> 定义了文档的信息
<title> 定义了文档的标题
<base> 定义了页面链接标签的默认链接地址
<link> 定义了一个文档和外部资源之间的关系
<meta> 定义了HTML文档中的元数据
<script> 定义了客户端的JavaScript脚本
<style> 定义了HTML文档的CSS样式

2.2 <body>

3. 文本/段落标签

与文本相关的标签，也是用得最多的标签。如标题、字体、段落、换行等。

标签 描述
<h1 - h6> 标题
<hr> 水平线分隔。无结束标签
<p> 段落
<br> 换行。无结束标签
标签 描述
<b> 定义粗体文本
<em> 定义着重文字
<i> 定义斜体字
<small> 定义小号字
<strong> 定义加重语气
<sub> 定义下标字
<sup> 定义上标字
<ins> 定义插入字
<del> 定义删除字
<cite> 定义引用、引证

至于font以及对齐方式等，一般在css中设置

4. 锚点/链接

4.1 <a>

Anchor Element：锚元素，<a> 标签定义超链接，用于从一个页面链接到另一个页面。

<a> 元素最重要的属性是href属性，它指定链接的目标。

<a href="https://nitgod1996.github.io/">进入宁理大神1996的快乐小屋!</a>

可用于：

  • 打开新页面

  • 移动到文档指定位置（当目录用）。这需要用到PJAX或者使用插件，大致逻辑如下

    1. 在标题处建立一个锚点，并设置href

      <a class="anchor" href="#2-创建虚拟机" data-pjax-state="anchor">#</a>

    2. 目录出设置href属性与标题的href一致

      <a class="toc-link" href="#2-%E5%88%9B%E5%BB%BA%E8%99%9A%E6%8B%9F%E6%9C%BA" data-pjax-state=""><span class="toc-text"> 2. 创建虚拟机</span></a>

主要属性如下：

属性 值 描述
href URL 规定链接的目标 URL。

其他属性如下：

属性 值 描述
charset char_encoding HTML5 不支持。规定目标 URL 的字符编码。
coords coordinates HTML5 不支持。规定链接的坐标。
download filename 指定下载链接
hreflang language_code 规定目标 URL 的基准语言。仅在 href 属性存在时使用。
media media_query 规定目标 URL 的媒介类型。默认值：all。仅在 href 属性存在时使用。
name section_name HTML5 不支持。规定锚的名称。
rel alternate author bookmark help license next nofollow noreferrer prefetch prev search tag 规定当前文档与目标 URL 之间的关系。仅在 href 属性存在时使用。
rev text HTML5 不支持。规定目标 URL 与当前文档之间的关系。
shape default rect circle poly HTML5 不支持。规定链接的形状。
target _blank _parent _self _top framename 规定在何处打开目标 URL。仅在 href 属性存在时使用。_blank：新窗口打开。_parent：在父窗口中打开链接。_self：默认，当前页面跳转。_top：在当前窗体打开链接，并替换当前的整个窗体(框架页)。
type MIME_type 规定目标 URL 的 MIME 类型。仅在 href 属性存在时使用。 注：MIME = Multipurpose Internet Mail Extensions。

4.2 <link>

<link> 主要用于外部资源的联系。最常见的用途是链接样式表。

如：

<link rel="stylesheet" href="css/style.css" type="text/css" media="all">

link只能写在头部。属性和<a>差不多

5. <img>

引入图片的标签

<img loading="lazy" src="smiley-2.gif" alt="Smiley face" width="42" height="42">

主要属性如下

属性 值 描述
loading eager：立即加载 lazy：延迟加载 指定浏览器是应立即加载图像还是延迟加载图像。
alt text 如果加载不出，规定图像的替代文本。
src URL 规定显示图像的 URL。
height pixels 规定图像的高度。
width pixels 规定图像的宽度。

其实heigth和width均可在CSS中设置，真正重要的也就src和alt属性。

java项目的默认路径一般是src同名文件夹

7. <div>/<span>无样式标签

<div>和<span>没有特殊定义。

<div>和<span>是最常用到的两个标签，因为html大部分时候仅仅写个文本就够了，排版、样式都是由CSS完成的。

<div>和<span>的唯一区别是

  • <div>是块级元素，单独成一行（也就是会自动换行）。适合做盒子或者对网页布局。
  • <span>是内联元素，不会自动换行。可以做文本容器.

一般用法是以<div>包裹<span>

8. <form>表单（接收用户输入）

HTML 表单用于收集不同类型的用户输入。以<form>标签包裹，里面设置如：文本域、下拉列表、单选框、复选框等。

元素包含一个或多个如下的表单元素：
  • input：规定了用户可以在其中输入数据的输入字段。
  • textarea：定义一个多行的文本输入控件。
  • button：按钮
  • select：下拉框
  • option：下拉列表的选项，类似<ul>和<li>的关系
  • optgroup：把相关的选项组合在一起。如图
  • fieldset：
  • label：为input元素标注，如图Male和Female就是`label
<form action="#" method="get">
    用户名：<input name = "username"><br>        
    密码：<input name="password"><br>            <!--请求参数-->
    <input type="submit" value="登录">
</form>

主要属性如下：

属性 值 描述
accept MIME_type HTML5 不支持。规定服务器接收到的文件的类型。（文件是通过文件上传提交的）
action URL 规定当提交表单时向何处发送表单数据。
method get/post 规定用于发送表单数据的 HTTP 方法。
name text 规定表单的名称。
target _blank/ _self /_parent /_top 规定在何处打开 action URL。

8.1 <input>

<input> 标签规定了用户可以在其中输入数据的输入字段。

<input> 元素在 <form> 元素中使用，用来声明允许用户输入数据的 input 控件。

主要属性如下：

属性 值 描述
name text name 属性规定 元素的名称。
readonly readonly readonly 属性规定输入字段是只读的。
size number size 属性规定以字符数计的 元素的可见宽度。
value text 指定 元素 value 的值。
type button checkbox image text 等 type 属性规定要显示的 元素的类型。

输入字段可通过多种方式改变，取决于 type 属性。type主要属性如下

  • button：定义可点击的按钮（通常与 JavaScript 一起使用来启动脚本）。

    <input type="button" value="点我" onclick="msg()">

    value - 按钮上显示的文字。onclick - 点击按钮事件执行的JavaScript代码

  • submit：定义提交按钮。**input除了button，任何需要提交的都需要用到submit**

  • checkbox：定义复选框。

    • name：name属性相同的单/复选框为一组
    • value：指定被选中后提交的值
    • checked：一开始默认选中的项
     <input type="checkbox" name="vehicle[]" value="Bike"> 自行车<br>
  <input type="checkbox" name="vehicle[]" value="Car"> 小轿车<br>
  <input type="checkbox" name="vehicle[]" value="Boat"> 船<br>
  <input type="submit" value="提交">

    name相同的复选框为一组。

  • radio：定义单选按钮。

      <input type="radio" name="gender" value="女"> 女<br>
  <input type="submit" value="提交">

  • file：定义文件选择字段和 “浏览…” 按钮，供文件上传。

      选择一个文件: <input type="file" name="img">
  <input type="submit">

  • password：定义密码字段（字段中的字符会被遮蔽）。

    • placeholder：指定输入框的提示信息，输入时会自动消失
    • value：提示信息，不会自动消失

  • text：默认。定义一个单行的文本字段（默认宽度为 20 个字符）。

      Email: <input type="text" name="email" placeholder="输入邮箱"><br>
  Password: <input type="password" name="pwd" maxlength="8" placeholder="输入密码"><br>
  <input type="submit">

8.2 <label>标签文本绑定

label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。

<label> 标签的 for 属性应当与相关元素的 id 属性相同。

  <label for="male">Male</label>
  <input type="radio" name="sex" id="male" value="male"><br>
  <label for="female">Female</label>
  <input type="radio" name="sex" id="female" value="female"><br><br>

如图，点击Male文本即可选中

8.3 <select>下拉框

<select> 元素用来创建下拉列表。不需要被<form>包裹

<select> 元素中的<option>标签定义了列表中的可用选项。

<select>
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="opel">Opel</option>
  <option value="audi">Audi</option>
</select>

主要属性如下：

属性 值 描述
disabled disabled 当该属性为 true 时，会禁用下拉列表。
name text 定义下拉列表的名称。
requiredNew required 规定用户在提交表单前必须选择一个下拉列表中的选项。
size number 规定下拉列表中可见选项的数目。

<option>主要属性如下

属性 值 描述
disabled disabled 规定此选项应在首次加载时被禁用。
selected selected 规定选项（在首次显示在列表中时）表现为选中状态。
value text 定义送往服务器的选项值。

8.4 <textarea>

<textarea> 标签定义一个多行的文本输入控件。

和<input type="text">不同，textarea一般用来接收整段的文字

<textarea rows="10" cols="30">
我是一个文本框。
</textarea>

其主要属性如下：

属性 值 描述
autofocusNew autofocus 规定当页面加载时，文本区域自动获得焦点。
rows number 规定文本区域内可见的行数。
cols number 规定文本区域内可见的宽度。
disabled disabled 规定禁用文本区域。
maxlengthNew number 规定文本区域允许的最大字符数。
name text 规定文本区域的名称。
placeholderNew text 规定一个简短的提示，描述文本区域期望的输入值。
readonly readonly 规定文本区域为只读。
requiredNew required 规定文本区域是必需的/必填的。

可在CSS中设置文本框不可拉伸

9. <table>表格

表格也可以用来页面布局

表格由 <table> 标签来定义。每个表格均有若干行（由<tr>标签定义），每行被分割为若干单元格（由 <td> 标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

<table border="1">
    <tr>
        <td>row 1, cell 1</td>
        <td>row 1, cell 2</td>
    </tr>
    <tr>
        <td>row 2, cell 1</td>
        <td>row 2, cell 2</td>
    </tr>
</table>
