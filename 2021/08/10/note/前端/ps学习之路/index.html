# 7. 存储格式的区别 (导出图片)

  1. PSD：ps 格式，含所有编辑内容

  2. JPEG：在线发布的照片的最佳格式

  3. PNG：包含透明度的图像

    1. 魔棒：消除背景。

    2. delete：

    3. Ctrl+D：取消选择；或者选择→取消选择

    1. 存储为 PNG

# 8. 调配颜色

图像→调整→

  1. 亮度 / 对比度：

    1. 亮度：增强高光
    2. 对比度：让亮的地方更亮，暗的更暗

  2. 饱和度

    1. 自然饱和度：暗色的强度
    2. 饱和度：所有颜色的强度，也就是鲜艳程度

  3. 滤镜

    1. 图像→调整→照片滤镜：设置普通滤镜，复古、暖色、冷色等

    2. 滤镜→转为智能滤镜；滤镜→风格化→可转为油画、浮雕等

      设置好后，如果想更改，则双击油画滤镜图层

    3. 还原：编辑→还原

  4. 阴影 / 高光

    1. 阴影：阴影部分的颜色强度（一般照片的背景）
    2. 高光：高光部分的颜色强度（一般是聚焦的部分）

  5. 黑白

# 3.2 叠加模式

首先，我们把背景成为基色，把图片成为混合色，把最终结果成为结果色。

如下：

基色：

# 组合模式

  • 正常：就是普通覆盖形式，只有不透明度会影响。大多数情况都是选用正常

    • 不透明度 = 100%：结果色 = 混合色
    • 不透明度 = 0%：结果色 = 基色

  • 溶解：混合色会以点状喷雾式混合在基色上，实际没什么卵用

# 加深模式

  • 变暗：通过比较基色和混合色，然后选择较暗的一方做为结果色

  • 正片叠底：将基色和混合色复合。白 + 彩 = 彩；黑 + 彩 = 黑；彩 1 + 彩 2 = 彩 3

    • 完全消灭基色中的白色
    • 除白色以外的任何颜色都会通过正片叠底模式让基色变暗
    • 任何颜色和黑色正片叠底，都只会得到黑色
    • 任何颜色和白色正片叠底，都会保持不变

    <font color=red> 正片叠底是这组中用得最多的模式 </font>

  • 颜色加深：保留画面中的白色区域，其他区域通过增加对比度的方式使基色变暗，再和混合色混合。

  • 线性加深：线性加深的效果和正片叠底差不多，但是比正片叠底要强烈的多，可以理解成正片叠底和颜色加深的结合体。

  • 深色：深色模式和变暗模式差不多，都是比较基色和混合色，谁暗谁就是结果色。和变暗模式不同的是，深色模式比较基色和混色色的所有通道之和，谁的数值小，谁显示，并不会产生第三种颜色。

# 三、美术理论基础

# 1. 透视

# 1.1 透视基本概念

https://baijiahao.baidu.com/s?id=1653262866067094917&wfr=spider&for=pc

  • 一点透视：所有延长线汇聚于一个点。（平视正方体一个面）视线的尽头一个点，多用于通道。

  • 两点透视：所有延长线汇聚 2 个点。（水平看正方体的直角边）

  • 三点透视：延长线汇聚于 3 个点（俯视 / 仰视正方体直角边）

# 1.2 三种透视关系共存

一张图并非只有一种透视关系，而是三种透视可以共存。

一般来说三点透视可以独立存在，而一点透视和两点透视需要所有延长线汇聚点在同一条线，（对摄像机来说是同一平面）

三种透视是根据摄像机的角度而变换的

  • 一点透视：摄像机正对正方体面
  • 两点透视：摄像机水平 / 垂直转动，视线与轴线平行
  • 三点透视：摄像机往另一个角度转动（如水平转→垂直转），焦点慢慢汇聚于正方体的角，再慢慢离开
