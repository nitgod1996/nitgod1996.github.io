# 安装

安装了 anaconda, 直接使用命令行： conda install jupyter notebook

使用命令行： pip install jupyter

# 打开

使用命令行： jupyter notebook

# 关闭

在命令符提示窗口：ctrl+C 停止服务，双击则跳过确认

# 简要说明

  1. 选择 markdown 时，可编辑标题和文本

  1. 使用快捷键 shift+enter : 执行单元格代码并移动到下一个单元格 run cell,select below

                  ctrl+enter : 运行当前单元格代码 run selected cells
              
              alt+enter : 运行当前单元格代码并创建下一个单元格 run cell and insert below

  1. 选择单元格，enter 进入编辑格式，esc 进入命令格式。 进入命令格式后，H-- 查看所有快捷键，M-- 切换成 markdown 单元，Y-- 切换成代码单元，B-- 在本单元的下方增加一个单元 。

  2. 在 Jupyter Note 的 markdown 单元格内可以使用 latex 语法进行数学公式编辑。

    文本行中插入数学公式用一对 $ ;

    文本行中插入数学块用一对 $$.

  1. 接收 HTML 代码，可以在 markdown 单元格内实现更加丰富的样式，添加图片等。

  2. 可以将 notebook 导出为多种格式 --HTML, Markdown, ReST, PDF (通过 Latex), Raw Python.

  3. 魔术关键字 http://ipython.readthedocs.io/en/stable/interactive/magics.html

  4. 幻灯片的制作 https://www.jianshu.com/p/86117613b7a6

  5. 查看.ipynb 文件的方法

①GitHub 中直接打开 ②可以把 .ipynb 文件对应的 github 之类的下载链接复制到 https://nbviewer.jupyter.org/ 中查看 ③安装 anaconda, 把 .ipynb 文件复制到 C：Users\name 目录下面，打开 Jupyter notebook. 或 cmd, 先 cd 到目标文件夹再 jupyter notebook

  1. 运行 python 文件：

①%run file.py ②！python test.py

导入外部代码： % load test.py

  1. 为 Jupyter Notebook 添加目录 点击打开链接

  2. 使用 matplotlib： % matplotlib inline
