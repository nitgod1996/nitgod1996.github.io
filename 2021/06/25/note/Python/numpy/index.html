# 一、 常用函数

注意：文本以 array 代表 np.array 类型的对象

# 1. 创建

# 1.1 自定义创建

x = np.array([[1, 2],
              [2, 3],
              [3, 4]])
array=np.array(x)#定义一个numpy类型矩阵
np.array(x,dtype=np.int)#括号里直接放数组，则是把list类型转换成numpy
np.array(x>0,dtype=np.int)#也可以放表达式，输出结果是bool型的numpy，x是数组

指定形状创建

np.zeros_like(x)#返回形状和x一致的矩阵，元素全部为0
np.zeros((1,2)) #输出形状为shape(1,2)的全是0的矩阵

循环形式创建

 x=np.arange(-5.0,5.0,0.1)
#在-5.0到5.0以0.1为单位(左闭右开)生成numpy数组[-5.0，-4.9....。4.9]

# 1.2 随机创建

随机抽取数创建：

np.random.choice(list,n)#从list中随机抽取n个元素组成array
np.random.choice(a,b)#从[0,a)中随机抽取b个元素组成array

生成随机数创建：

a=np.random.rand(100000)    #随机创建一个长10000的向量
a2=np.random.rand(100,100)    #随机创建一个100*100的向量

rand 生成的随机数是 < 1 的

# 2. np.array 的性质

np.array 是 n 维向量（矩阵），在 Python 中，多维向量类型常用的主要有 3 个：list、DataFrame 和 np.array。后两者显然更适合科研数据处理，其中，dataframe 主要倾向于数据类型、数据格式等处理；np.array 主要倾向于矩阵运算

# 3. 矩阵运算

乘法：

  • np.dot(x,y) ：矩阵乘法（行 X 列）， x 和 y 不一定是 np.array 类型
  • x*y ：对应位置相乘。1. x 必须是 np.array 类型。2. x 和 y 必须形状相同

加减法：

  • x+y / x-y ：对应位置相加 / 键、减

逻辑运算：

  • x<2 ：每个元素和 2 比较，返回一个 bool 类型矩阵

    x = np.array([[1, 2],
              [2, 3],
              [3, 4]])
print(x>2)
"""
[[False False]
 [False  True]
 [ True  True]]
"""

求和：

  • np.sum(x) ：所有元素相加

# 4.4 查

和 dataframe 的 .iloc[] 用法差不多

  • 取单个元素： x[1,1] ：# 矩阵第 2 行第 2 列的元素

  • 取行向量：

    • x[1] ：第 2 行的向量
    • x[1,...] ：第 2 的向量
    • x[[1,2]] ：第 2,3 行的向量

  • 取列向量：

    • x[:,1] ：矩阵第 2 列的向量（从 0 开始）
    • x[...,1] ：第 2 列向量
    • y[:,[1,2]] ：矩阵第 2,3 列的向量

  • 获取矩阵形状：

    • array.shape[n] ：返回矩阵第 n+1 维的长度

      如： x[0] 返回 3， x[1] 返回 2。

      ​ 最外面那层 [] 包裹的就是第 1 维（行）

      ​ 里面一层 [] 是第 2 维（列）

      ​ 再里面是第 3 维

    • np.shape(x) ：返回矩阵 x 的各维度长度，此例返回 **(3, 2)**

    • np.ndim(x) ：返回矩阵的维数，此例返回 2

  • 获取特殊值的索引：

    • array.argmax() ：返回向量最大值的索引
    • np.argmax(array,axis=n) ：返回向量在第 n+1 维最大值的索引，可不写
